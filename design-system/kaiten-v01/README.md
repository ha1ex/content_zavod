# Kaiten Design System V01 — источник истины

Эта папка — **канонический источник истины** дизайн-системы Kaiten V01.
Все производные артефакты (токены в коде, prompt-методички для LLM, skill для Claude)
выводятся из материалов в этой папке. При любом расхождении приоритет — за оригиналом отсюда.

## Состав

| Файл | Что это |
| --- | --- |
| `design-system.html` | Общий гайд по дизайн-системе (цвета, типографика, сетки, токены, состояния компонентов). Самостоятельная printable-страница A4 landscape. |
| `site-construction-guide.html` | Руководство по верстке лендингов: layout-правила, ритмы, состояния, паттерны секций (header, hero, FAQ, отзывы, CTA, footer). |
| `site-construction-guide.pdf` | PDF-версия руководства по верстке (для печати/просмотра без браузера). |
| `pages/page-01.png … page-16.png` | Постраничные скриншоты PDF — удобно для multimodal-LLM, ревью и быстрого визуального reference. |

Откройте HTML-файлы локально в браузере — они стилизованы под A4 и читаются как презентация.

## Где используются производные

Все ниже — это **выжимки** из материалов выше; их обновляют только если изменился source.

| Где | Что | Назначение |
| --- | --- | --- |
| `packages/ui/src/tokens.css` | `@theme`-блок с CSS-переменными | Runtime-токены для всех TSX-компонентов (Tailwind v4). |
| `packages/harness/src/prompts/design-system-kaiten.md` | Markdown-методичка | Подмешивается в system prompt LLM при генерации лендингов. |
| `.claude/skills/design-system-kaiten-v01/SKILL.md` | Skill | Активируется в Claude Code, когда задача касается дизайна/верстки Kaiten. |

## Как пользоваться

- **Делаете новый лендинг или компонент** — сначала откройте `design-system.html` и `site-construction-guide.html`; они дают полный контекст за 10 минут.
- **Меняется фирстиль (новые цвета, шрифт, ритмы)** — правьте оригиналы здесь, затем синхронизируйте `tokens.css`, prompt и SKILL. Не наоборот.
- **LLM-задачи** — multimodal-агентам можно скармливать `pages/*.png` напрямую как визуальный reference.

## Ограничения V01

- В Figma-файле есть зона `Bloks` (экспериментальные блоки) — её материалов здесь намеренно нет. Не используйте их без явной просьбы.
- Все макеты — десктоп `1920px`, планшет `768px`, мобайл `360px`. Другие брейкпоинты — экстраполируйте по сетке.

## Версионирование

Подпапка называется `kaiten-v01`. При следующей крупной ревизии системы (новые токены, переименование слоёв) — создавайте `kaiten-v02/` рядом, а не правьте этот.
