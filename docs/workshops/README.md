# Воркшопы: обучение команды «Контент-завод Кайтен»

Материалы для обучения нетехнической маркетинг-команды работе с репозиторием:
генерация лендингов, доработка репозитория, передача готового разработке.

Формат: **3 интенсива + домашки между ними**. Только лендинги (статьи — будущий
отдельный трек).

## Как открыть презентацию

Просто открой `program.html` в браузере — двойной клик по файлу или перетащи в окно
браузера. Сервер не нужен, всё в одном файле.

Навигация:
- **→ / Space / ↓** — следующий слайд;
- **← / ↑** — предыдущий;
- точки справа — переход к любому слайду;
- **Home / End** — в начало / в конец.

Для показа на проекторе разверни браузер на весь экран (F11 / полноэкранный режим).

## Как пользоваться домашками

Каждую домашку можно отправлять команде **до** соответствующего воркшопа: внутри —
чек-лист простым языком, точные команды и критерий «сдано». Цель — приходить на
воркшоп уже подготовленными, чтобы время тратить на практику, а не на установку.
