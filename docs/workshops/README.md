# Воркшопы: обучение команды «Контент-завод Кайтен»

Материалы для обучения нетехнической маркетинг-команды: как самим делать лендинги,
улучшать «завод» (правила сборки) и передавать готовое разработке — **общаясь с
агентом по-русски, без команд и кода руками**.

Формат: **3 интенсива + домашки между ними**. Только лендинги (статьи — будущий
отдельный трек).

## Как открыть презентацию

Откройте `program.html` в браузере — двойной клик или перетащите в окно. Сервер не
нужен, всё в одном файле.

Навигация:
- **→ / Space / ↓** — следующий слайд, **← / ↑** — предыдущий;
- точки справа — переход к любому слайду; **Home / End** — в начало / в конец;
- для показа разверните браузер на весь экран.

## Принципы материалов

- **Никаких команд.** Везде, где раньше был терминал, теперь русская фраза агенту
  («покажи лендинг в браузере», «поправь картинку и покажи, как стало»).
- **Сначала понимание.** Отдельные слайды объясняют, из чего состоит репозиторий и
  как идёт сборка по шагам (пайплайн), чтобы команда видела, что и где чинить.
- **Учим улучшать завод.** Ядро программы — что делать, когда лендинг вышел «не очень»:
  читать «красные блоки», дорабатывать правило нужного шага и сохранять в проект.
  Для этого есть **универсальный промпт для правок**.
