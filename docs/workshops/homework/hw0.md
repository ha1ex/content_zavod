# Домашка 0 — «Выравнивание базы»

**Перед воркшопом 1. Делается дома, по шагам. Времени — 30–60 минут.**

Цель простая: чтобы на первом воркшопе у всех на экране открылся работающий проект.
Если застрял на любом шаге — не мучайся, напиши в общий чат, поможем. Это нормально.

---

## Шаг 1. Доступ к репозиторию

1. Если нет аккаунта GitHub — заведи на [github.com](https://github.com) (бесплатно).
2. Пришли свой ник в GitHub в общий чат — тебя добавят в репозиторий.
3. Прими приглашение (придёт на почту или в уведомления GitHub).

**Готово, когда:** ты видишь репозиторий в списке своих на GitHub.

## Шаг 2. Установить Claude Code

1. Установи Claude Code (десктоп-приложение или через инструкцию команды).
2. Войди под своим аккаунтом.

> Claude Code — это наш главный инструмент. Внутри него мы будем «разговаривать» с агентом
> простым русским текстом, а он будет делать работу. Никакого кода руками.

**Готово, когда:** Claude Code открывается и просит выбрать папку/проект.

## Шаг 3. Скачать проект (клонировать репо)

Терминал не нужен — используем приложение с кнопками:

- **Conductor** (рекомендуем, если на Mac) — или **GitHub Desktop**.
- Нажми «Clone repository» → выбери наш репозиторий → выбери папку на компьютере → Clone.

**Готово, когда:** на компьютере появилась папка проекта с файлами.

## Шаг 4. Установить зависимости

Один раз, чтобы проект «ожил». В Claude Code (или в терминале приложения) выполни:

```
pnpm install
```

Дождись, пока закончится (может занять пару минут).

**Готово, когда:** команда завершилась без красных ошибок.

## Шаг 5. Запустить проект и открыть в браузере

```
pnpm dev
```

Затем открой в браузере:

```
http://localhost:3000
```

Полистай — там уже есть готовые лендинги. Это и есть наш «завод».

**Готово, когда:** страница открылась и показывает интерфейс.

## Шаг 6. Прочитать редполитику

Открой файл `wiki/brand/redpolitika.md` и прочитай. Это правила языка и тона —
по ним агент проверяет тексты. Глубоко вникать не надо, достаточно «о чём это».
