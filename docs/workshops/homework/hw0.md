# Домашка 0 — «Подготовьте рабочее место»

**Перед воркшопом. Делается дома. Времени — 30–60 минут.**

Цель: чтобы на воркшопе у всех на экране открылся работающий проект.
Никаких команд печатать не нужно — где надо, вы просто пишете агенту по-русски,
что сделать. Застряли — напишите в общий чат, поможем. Это нормально.

---

## Шаг 1. Аккаунт GitHub

1. Если нет аккаунта — заведите на [github.com](https://github.com) (бесплатно).
2. Пришлите свой ник в общий чат — вас добавят в проект.
3. Примите приглашение (придёт на почту или в уведомления GitHub).

**Готово, когда:** проект виден в списке ваших на GitHub.

## Шаг 2. Установите Claude Code

1. Установите [Claude Code](https://claude.com/claude-code) — это наш главный инструмент.
2. Войдите под своим аккаунтом.

> Внутри Claude Code вы будете «разговаривать» с агентом обычными словами по-русски,
> а он будет делать всю техническую работу. Код руками — не нужен.

**Готово, когда:** Claude Code открывается.

## Шаг 3. Скопируйте проект себе

Через **Conductor** ([conductor.build](https://conductor.build)) или прямо в **Claude Code**:
нажмите «склонировать репозиторий» и вставьте ссылку на наш проект
([github.com/ha1ex/site_zavod](https://github.com/ha1ex/site_zavod)).

«Склонировать» = скопировать проект себе на компьютер. Кнопками, без терминала.

**Готово, когда:** проект скопировался и открылся.

## Шаг 4. Запустите и посмотрите лендинги

Откройте проект в Claude Code и напишите в чат:

> **«Запусти проект и покажи мне лендинги в браузере»**

Агент всё сделает сам и откроет страницу. Полистайте — там уже есть готовые лендинги.
Это и есть наш «завод».

**Готово, когда:** в браузере открылись лендинги.

## Шаг 5. Познакомьтесь с проектом

Напишите агенту:

> **«Проведи экскурсию по проекту — из чего он состоит и что где лежит»**

Не нужно ничего читать по файлам и путям. Просто послушайте агента — этого хватит,
чтобы прийти на воркшоп с пониманием.

## Шаг 6. Заведите аккаунт на Vercel (2 минуты)

На воркшопе будем публиковать ваш лендинг в интернет. Для этого:

1. Зайдите на [vercel.com](https://vercel.com).
2. Нажмите «Sign Up» → **«Continue with GitHub»** (входите через свой GitHub-аккаунт).
3. Больше ничего делать не нужно — остальное на воркшопе.
