Контент-завод Кайтен:
от нуля до лендинга

Как нетехническая команда — писатели, редакторы, маркетологи, дизайнеры — сама делает лендинги. Разговаривая с агентом по-русски. Без знания кода.

Цель программы

Что вы сможете сами

1

Генерить лендинги

От брифа до готовой страницы — диктуя задачу агенту простым текстом. Агент пишет код, вы держите смысл.

2

Дорабатывать репозиторий

Править брифы, обновлять контент, добавлять уроки в общую базу знаний — без разработчика.

3

Передавать разработке

Отдавать готовый, проверенный лендинг разработчикам одним действием — handoff.

Домашка 0 · перед стартом · self-paced

Выравнивание базы

Чтобы на первом воркшопе все были на одной странице — буквально. Делается дома, по шагам.

  • Получить доступ к GitHub-репозиторию
  • Установить Claude Code
  • Склонировать репо через Conductor или GitHub Desktop
  • Выполнить pnpm install
  • Запустить pnpm dev и открыть localhost:3000
  • Прочитать редполитику wiki/brand/redpolitika.md
Сдано = скриншот работающего localhost:3000

  • Выбрать реальный продукт / возможность для лендинга

  • Прочитать wiki/marketing/brief-fields.md — что класть в бриф

Сдано = готовая папка inputs/<slug>/ с материалами
Воркшоп 2 · ~2.5 часа · интенсив

Фабрика ТЗ + генерация своего лендинга

Блок A · фабрика ТЗ
  • /kaiten-intake на своём сырье
  • Ревью ТЗ на странице /intake/<slug>
  • Редполитика на практике: язык, без лозунгов, боли как улучшение
  • Одобрить ТЗ
Блок B · генерация + ревью
  • /kaiten-generate от своего брифа
  • Читать подсказки проверок, чинить по кругу
  • Превью /landings/<slug>, визуальное ревью
  • Страница одобрения /approve/<slug>
Почему агент иногда «не пускает»: блок-картинка должна быть из домена продукта, язык — по редполитике, аудитория — покрыта. Это встроенный редактор, который держит планку.
Домашка 2 · перед воркшопом 3

Доведите свой лендинг до одобрения

  • Пройти весь круг: intake → generate → превью
  • Починить все замечания проверок
  • Одобрить на /approve/<slug>
  • Выписать блокеры и вопросы, которые возникли
Сдано = лендинг в статусе approved + список вопросов

Почему агент иногда блокирует

Правила качества

Язык — редполитика

Англицизмы по-русски (страница, а не «лендинг»). Без лозунгов и штампов («революционный», «в 10 раз»). Боли — как управляемое улучшение, не обвинение.

Смысл — гейты

Картинка-блок строго из домена продукта. Аудитория должна быть покрыта по сценариям. Структура — по проверенному макету.

Это не ошибки и не баги. Это встроенный редактор, который не даёт выпустить слабую страницу. Подсказку всегда видно — и всегда понятно, что починить.
